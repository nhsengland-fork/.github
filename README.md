# nhsengland-fork

This Organization simply serves as a host for the forks of all public repos in the nhsengland organization and was created as a response to the following blog post

# What this is
- A backup of all publicly available repos for the sake of the open source community in the hope that if the public repos become private then up to the last day of work will have been preserved in a fork network
- Automated such that repos will have their forks synced once a day (midnight bst)

# What this isn't
- An attempt to infringe on the rights or licenses of NHS England
- An active development organization, I do not plan to work on the repos myself within this organization and PRs made to these forks as well as any issues opened will not be worked on currently
